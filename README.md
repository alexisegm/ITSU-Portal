# ITSU-Portal

Plataforma institucional de gestión académica diseñada para centralizar el seguimiento del rendimiento estudiantil, la administración de usuarios y la generación de reportes automáticos.

## 🚀 Descripción del Proyecto
ITSU-Portal es un sistema web robusto orientado a optimizar la interacción entre la institución y sus estudiantes. El sistema permite la consulta de boletines de calificaciones, gestión administrativa y automatización de procesos, garantizando seguridad y escalabilidad mediante una arquitectura basada en la nube.

## 🛠 Tecnologías Utilizadas

### Backend
* **Flask (Python):** Framework ligero y potente para la creación de la API RESTful.
* **MongoDB Atlas:** Base de datos NoSQL alojada en la nube para el almacenamiento seguro de perfiles, notas y registros institucionales.
* **python-dotenv:** Gestión segura de variables de entorno y credenciales de infraestructura.

### Frontend
* **Vanilla JavaScript (ES6+):** Lógica de aplicación basada en componentes modulares sin dependencias pesadas.
* **HTML5 y CSS3 (Tailwind CSS):** Diseño responsivo, moderno y altamente personalizable.
* **Fetch API:** Comunicación asíncrona para la actualización de datos en tiempo real sin recargar la página.

## ⚙️ Características Principales
* **Sistema de Autenticación Seguro:** Gestión de sesiones mediante cookies de sesión y validación de credenciales en el servidor.
* **Panel de Control Dinámico:** Interfaz de usuario (SPA) que se adapta según el rol (Estudiante o Administrador).
* **Visualización de Rendimiento:** Consulta de notas organizada por periodos académicos con generación de reportes.
* **Seguridad Profesional:** Implementación de prácticas de desarrollo seguro (CORS, variables de entorno ocultas y control de acceso).
